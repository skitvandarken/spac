Cloud

Infraestrutura como serviço (IaaS) dedicada à Internet das Coisas e Soluções Off-Grid

Combinando poder de computação, armazenamento e conectividade.

  • Network Server IoT Integrado

    Plataforma LoRaWAN completa com Network Server, Application Server e Gateway Bridge pré-configurados.

  • Containers e VMs

    Virtualização leve com containers LXC e VMs na nuvem para isolamento de aplicações e projectos para grandes projectos de IoT.

  • Gestão de Dispositivos

    Cadastro, provisionamento e monitoramento de dispositivos LoRaWAN através de interface web intuitiva.

  • Containers Gerenciados com GUI

    Gestão simplificada de containers com uma interface web intuitiva, escalabilidade automática e balanceamento de carga.

  • Segurança LoRaWAN

    Autenticação e criptografia end-to-end, chaves de sessão OTAA/ABP e controle de acesso por dispositivo.

  • Multi-tenancy

    Arquitetura multi-inquilino para suportar múltiplos clientes com isolamento completo de dados e aplicações.
